2021/07/24

Hello World

Rust 语言中最简单的一个Hello World程序

fn main() {
	println!("Hello Rust!");
}

在Rust中println!是一个宏(Marco)用于输出字符串
rust程序文件后缀为 rs 编译一个rust程序，可以使用 rustc helloworld.rs rustc编译器编译完程序文件后可以生成一个二进制文件
